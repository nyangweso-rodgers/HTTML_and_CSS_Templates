Columns

Offseting Columns

Grid columns can be offset in two ways: using responsive .offset-grid classes and margin utilities

Grid classes are sized to fit columns while margins are more useful for quick layouts where the offset width is variable.

Offset Classes

Move columns to the right using .offset-md-* class. These classes increase the left margin of a column by *columns

.col-md-4
.col-md-4 .offset-md-4
.col-md-3 .offset-md-3
.col-md-3 .offset-md-3
.col-md-6 .offset-md-3

Margin Utilities

With the move to flexbox, you can use margin utilities such as .me-auto to force sibling columns away from each other.

.col-md-4
.col-md-4 .ms-auto
.col-md-3 .ms-md-auto
.col-md-3 .ms-md-auto
.col-auto .me-auto
.col-auto
