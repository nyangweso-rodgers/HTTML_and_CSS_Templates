CSS Box Model

  • CSS Margins: properties are used to create space around elements, outside of any defined borders.
  • CSS Padding: used to create space around an element's content, inside of any defined borders.
  • CSS Height: used to set the height of an element
  • CSS Width: used to set the width of an element

Setting max-width

Using the max-width improve the browser's handling of small windows.

Note: If you for some reason use both the width property and the max-width property on the same element, and the value of the width property is larger than the max-width property; the max-width property will be used (and the width property will be ignored)
