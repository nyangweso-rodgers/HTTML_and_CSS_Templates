Grid Item

Grid Item Properties

  • The grid-column Property: defines on which column(s) to place an item. You define where the item will start, and where the item will end. To place an item, you can refer to line numbers, or use the keyword "span" to define how many columns the item will span.
  • The grid-row Property: defines on which row to place an item. You define where the item will start, and where the item will end. To place an item, you can refer to line numbers, or use the keyword "span" to define how many rows the item will span:
  • The grid-area Property: can be used as a shorthand property for the grid-row-start, grid-column-start, grid-row-end, and grid-column-end Properties.

3. The grid-area Property

You can use the grid-area proerty to specify where to place an item.

The syntax is:

grid-row-start / grid-column-start / grid-row-end / grid-column-end.

Item 8 will start on row-line 1 and column-line 2, and end on row-line 5 and column-line 6

Naming Grid Items

The grid-area property can also be used to assign names to grid items.

Named grid items can be referred to by grid-template-areas property of the grid container
