CSS - Text Properties

  • font-family
  • font-size: you can use numeric values in pixels or ems to manipulate font-size. the ems size is a relative size unit. it allows all major browsera to resize the text. the browser's default size is 16px. Note: ems = pixels/16
  • font-style: has 3 values:
    • normal
    • italic
    • oblique
  • font-weight
    • normal
    • bold
    • bolder
    • lighter
    • you can also define font-weight with a number from 100(thin to 900(thick
  • font-variant: allows you to convert your font to all small caps. The values include:
    • small
    • small-caps
    • inherit
  • text-align: specifies the horizontal alignment of texts in an element. The default value is left. Values include:
    • left
    • right
    • center
    • justify: each line is streteched so that every line has equal width, and the left and right margins are straight (as magazin newpapers)
  • vertical-align: values include:
    • top
    • middle
    • bottom
    Other vertical-align properties include:
    • baseline
    • sub
    • super
    • %
    • px (or cm, pt)
  • text-decoration: values include:
    • none - default value, it defines a normal text
    • inherit: from the parent element
    • overline
    • underline
    • line-through
    • blink : this value has been depreciated by most browsers
  • text-indent: specifies how much horizontal space should be left before the beginning of the first line of text. values include:
    • length (px, pt, cm, e.t.c.,), 100px
    • 100%
    • inherit
    Remark: negative values are allowed. the first line will be indented to the left if the values is negative
  • text-shadow: adds shadow to the text.
    • the first value defines the distance of the shadow in x direction
    • the second value sets the distance in the y (vertical) direction
    • the third value defines the blur of the shadow
    • the fourth value sets the color.
  • text-transform: specifies how to capitalize an element's text. e.g., it can be used to make text appear with each word capitaized. values include:
    • all-uppercase
    • all-lowercase
    • none
  • letter-spacing: specifies the space between characters. the values can be set as:
    • normal: defines the default style with no extra characters between
    • length: defines an extra space between characters using measurement units like px, pt,cm, mm
    • inherit:
  • word-spacing:specifies spaces between words in a text. values include:
    • normal
    • 10pt length
    • inherit
  • white-space:specifies how a white-space inside an element is handled. values can be set as
    • normal
    • inherit
    • nowrap: makes the text continue on the same line until a break tag is encountered, and also collapse into a single whitespace
    Remark: other supported values include:
    • pre : text will only wrap on line breaks and white space
    • pre-line : text will wrap where there is a break in code, but extra white space is still ignored.
    • pre-wrap : text will wrap when necessary, and on line breaks. behaves as the pre value, except that it add extra line breaks to preent the tet breaking out of the element box
